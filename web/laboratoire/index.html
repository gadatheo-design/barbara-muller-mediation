Usage en soutenance

Comment utiliser ce laboratoire pour la présentation HKB

1. Montrer le prototype public

Commencer par le prototype mobile principal pour situer la version la plus directement testable en contexte de visite.

2. Ouvrir 2 ou 3 expériences web

Choisir quelques sketches p5.js courts pour faire sentir différentes questions de médiation: voisinage de couleur, immobilité, vocabulaire, retour du regard.

3. Compléter avec Processing

Montrer des captures d'écran ou des sketches locaux Processing pour ouvrir la dimension laboratoire, projection et installation.

4. Nommer les limites et les suites

Expliquer que ces essais restent des sketches exploratoires, puis ouvrir sur les possibilités futures: simplification, projection, version tablette, formats hybrides.

Cette version publique du laboratoire ne dépend d'aucun backend, n'utilise ni cookies, ni analytics, ni localStorage.
