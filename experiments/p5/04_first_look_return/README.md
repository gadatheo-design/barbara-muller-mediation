# Premier regard / retour

## Statut

Sketch exploratoire p5.js tres leger, pense comme outil de mediation plutot que comme instrument de mesure.

## Intention

Permettre de marquer un premier point d'attention puis un retour du regard sans recourir a un systeme de suivi oculaire.

## Interaction

Le premier appui marque un point initial, le second marque un retour. Une ligne discrete relie ensuite les deux positions.

## Relation a la mediation Barbara Muller

Cette piste prolonge le marquage A/B du prototype principal, mais de maniere plus calme, plus lisible et plus modeste.

## Ce que cela teste

Le sketch teste une forme de documentation du deplacement du regard qui reste qualitative et compréhensible pour le visiteur.

## Usage futur possible

Cette logique pourrait revenir dans un mode test simplifie, dans une version tablette accompagnee, ou dans une documentation de presentation HKB.
