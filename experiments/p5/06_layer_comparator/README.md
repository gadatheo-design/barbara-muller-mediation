# Comparateur de couches

## Statut

Sketch exploratoire p5.js autonome, concu comme outil de comparaison visuelle.

## Intention

Montrer qu'une surface picturale peut etre approchée par une lecture de couches et de traces sans etre totalement expliquee.

## Interaction

Le visiteur fait glisser un curseur vertical entre une surface abstraite et une lecture simplifiee de sa structure.

## Relation a la mediation Barbara Muller

Cette piste prolonge la tension entre experience sensible et explicitation analytique presente dans tout le projet.

## Ce que cela teste

Le sketch teste jusqu'ou une analyse visuelle peut aider le regard sans l'ecraser ni se substituer a l'oeuvre.

## Usage futur possible

Cette logique pourrait nourrir une version de demonstration, un support de mediation accompagnee, ou une diapositive interactive pendant la presentation HKB.
