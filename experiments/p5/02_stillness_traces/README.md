# Traces d'immobilite

## Statut

Sketch exploratoire p5.js autonome, sans backend.

## Intention

Transformer l'immobilite en geste actif plutot qu'en absence d'action.

## Interaction

La trace n'apparait que lorsque le pointeur reste presque immobile. Les mouvements rapides dissipent les marques.

## Relation a la mediation Barbara Muller

Cette experience rejoint les enjeux de regard lent et de temporalite deja presents dans les phases les plus contemplatives du projet.

## Ce que cela teste

Le sketch teste si une interaction minimale peut encourager une attention plus patiente sans recourir a un texte explicatif important.

## Usage futur possible

Cette logique pourrait nourrir une etape de parcours guide, une version tablette, ou une projection calme dans un contexte de presentation.
