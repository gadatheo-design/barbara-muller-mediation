# Carre a couleur fixe

## Statut

Sketch exploratoire p5.js autonome, runnable localement dans un navigateur.

## Intention

Montrer qu'une meme couleur peut etre percue differemment selon le champ coloré qui l'entoure.

## Interaction

Le visiteur fait glisser un carre dont la teinte reste strictement stable. Le halo autour du carre change avec la region traversée.

## Relation a la mediation Barbara Muller

Cette experience prolonge la question des relations de voisinage et de perception contextuelle deja presente dans le prototype mobile.

## Ce que cela teste

Le sketch teste la possibilite de faire sentir une relation colorée sans recours a une explication longue ni a une image d'oeuvre.

## Usage futur possible

Cette piste pourrait nourrir une interaction courte dans le prototype principal, une version tablette, ou un exercice de mediation autonome pendant une presentation.
