# Collective projection

## Statut

Sketch Processing exploratoire oriente projection et installation.

## Intention

Imaginer ce que pourrait devenir le projet si plusieurs attentions de visiteurs etaient visualisees collectivement.

## Interaction

Le sketch simule plusieurs visiteurs virtuels qui laissent des chemins, des points et des mots autour de trois foyers d'attention.

## Relation a la mediation Barbara Muller

Cette piste deplace le projet du cote d'une mediation collective et d'une possible installation de presentation HKB.

## Ce que cela teste

Elle teste la valeur d'une visualisation partagee : non pas mesurer scientifiquement, mais rendre visible une memoire collective de parcours et de mots.

## Usage futur possible

Cette logique pourrait nourrir une projection de soutenance, une installation speculative, ou une version publique plus collective du laboratoire.
