# Density resistance

## Statut

Sketch Processing exploratoire qui traduit la densite picturale en comportement interactif.

## Intention

Faire sentir qu'une masse sombre peut agir comme une resistance plutot que comme une simple forme.

## Interaction

Quand on dessine pres de la masse, le mouvement se freine, se courbe et s'epaissit.

## Relation a la mediation Barbara Muller

Le sketch prolonge les questions de densite, de contrepoids et de tension deja presentes dans plusieurs pistes du laboratoire.

## Ce que cela teste

Il teste comment une qualite picturale peut devenir une resistance sensible dans l'interaction.

## Usage futur possible

Cette logique pourrait nourrir un module d'atelier, une version installative, ou une explication plus incarnée de la notion de masse picturale.
