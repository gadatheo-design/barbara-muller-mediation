## Usage rapide

- si le besoin est oral : commencer par [00_PRESENTATION_PLAN.md](./00_PRESENTATION_PLAN.md)
- si le besoin est visuel : commencer par [01_SLIDE_OUTLINE.md](./01_SLIDE_OUTLINE.md) et [05_VISUAL_MATERIALS_CHECKLIST.md](./05_VISUAL_MATERIALS_CHECKLIST.md)
- si le besoin est une demo stable : commencer par [02_DEMO_SCRIPT.md](./02_DEMO_SCRIPT.md)
- si le besoin est de défendre le projet comme laboratoire : commencer par [03_ARGUMENTAIRE.md](./03_ARGUMENTAIRE.md)
- si le besoin est de comparer vite plusieurs formats après la démo : utiliser [06_GRILLE_COMPARAISON_HKB.md](./06_GRILLE_COMPARAISON_HKB.md)
- si le besoin est d'avoir un repere oral deja formule : utiliser [07_GRILLE_COMPARAISON_HKB_AIDE_MEMOIRE.md](./07_GRILLE_COMPARAISON_HKB_AIDE_MEMOIRE.md)
