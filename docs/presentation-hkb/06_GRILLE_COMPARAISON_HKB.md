# Grille de comparaison HKB

## Usage

Cette grille tient en **une page** et sert a comparer rapidement les formats montres pendant une demonstration ou juste apres la soutenance HKB. Elle ne remplace pas la matrice comparative longue ; elle sert plutot de support de relecture immediate.

Formats proposes pour la comparaison :

- prototype mobile guide
- sketch p5 `Carre de couleur fixe`
- low-tech `Calques transparents`
- Arduino `Table de matieres`

## Echelle rapide

- comprehension initiale :
  - faible
  - moyenne
  - forte
- retour a l'oeuvre :
  - faible
  - moyen
  - fort
- besoin de mediation orale :
  - faible
  - moyen
  - fort
