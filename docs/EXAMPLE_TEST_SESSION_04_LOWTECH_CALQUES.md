# Fiche d'exemple de session de test 04 — Low-tech Calques transparents

## Statut du document

Cette fiche est une **session d'exemple remplie** a partir de la documentation low-tech actuelle autour des `Calques transparents`. Elle montre comment documenter un essai de mediation sans electronique, centre sur les couches, les voiles et les contrepoids visibles dans une reproduction. Elle ne doit pas etre lue comme une preuve terrain exhaustive.

## Identification

- Date : exemple de session a completer lors d'un vrai essai
- Lieu : demonstration HKB, mediation en petit groupe, ou atelier rapide
- Personne qui observe / medie : etudiant·e ou mediateur·trice du projet
- Format teste : dispositif low-tech `Calques transparents`
- Version ou fichier concerne : `docs/lowtech/01_calques_transparents.md`
- Duree approximative : 4 a 6 minutes

## Question principale du test

- Que cherche-t-on a verifier dans cette session ?

Verifier si la superposition et le retrait de calques aident a comprendre une peinture comme construction de couches, de voiles et d'equilibres, sans passer par un ecran ni par une interface numerique.

## Contexte

- Type de situation :
  - demonstration HKB
  - test entre pairs
  - mediation en petit groupe
- Nombre de personnes presentes :
  - 1 a 4
- Public concerne :
  - pair, enseignant, visiteur curieux, groupe peu attire par le smartphone
- Conditions particulieres :
  - reproduction imprimee de l'oeuvre
  - feuilles transparentes preparees a l'avance
  - mediation orale breve

## Profil general du visiteur ou du groupe

- Age approximatif ou type de public :
  - tres large, adolescent a adulte
- Familiarite avec art abstrait :
  - variable
- Familiarite avec smartphone / dispositif :
  - sans importance
- Commentaire utile :
  - le format est particulierement utile quand on veut ralentir sans passer par un outil numerique

## Deroulement

- Comment l'entree dans l'experience a ete introduite :
  - une phrase du type : "Vous pouvez ajouter ou retirer des couches pour voir comment l'ensemble se tient autrement."
- Le visiteur a-t-il compris rapidement ce qu'il fallait faire :
  - oui, le geste de superposer ou d'enlever un calque est immediat
- Le dispositif a-t-il demande une explication orale supplementaire :
  - oui, mais courte, surtout pour relier le geste a la notion de construction picturale
- A quel moment principal une hesitation est apparue :
  - quand il faut formuler avec des mots ce que change une couche retiree
- A quel moment principal une comprehension ou un engagement est apparu :
  - lorsque l'enlevement d'un calque modifie nettement l'equilibre general ou la reserve

## Observation qualitative

### Comprehension

- comprehension immediate / tardive / fragile :
  - immediate pour l'action, solide apres une ou deux manipulations
- ce qui a ete compris :
  - qu'une peinture peut se lire comme un systeme de couches et de contrepoids
- ce qui est reste flou :
  - la precision historique ou technique du rapport entre calques et peinture reelle ; il faut rappeler qu'il s'agit d'une mediation simplifiee

### Rythme

- rythme trop rapide / juste / trop lent :
  - juste
- le visiteur a-t-il pris le temps :
  - oui, le geste invite naturellement a ralentir
- l'attention est-elle restee sur l'ecran, le materiau ou l'oeuvre :
  - sur la reproduction et le materiau, avec un passage plus direct vers l'oeuvre que dans un format d'ecran

### Geste ou action

- le geste ou l'action etait-il lisible :
  - tres lisible
- le changement produit etait-il perceptible :
  - oui, surtout si les calques sont peu nombreux et clairement distincts
- le moment de passage a l'etape suivante etait-il clair :
  - oui ; on peut enchainer vers une question orale ou vers le regard direct sur l'oeuvre

### Retour a l'oeuvre

- le retour a l'oeuvre a-t-il eu lieu :
  - oui
- si oui, de quelle maniere :
  - le visiteur revient plus facilement a la reproduction ou a l'oeuvre pour verifier ce qu'il croit avoir compris des couches et des reserves
- si non, pourquoi probablement :
  - si les calques sont trop demonstratifs ou trop nombreux, l'attention peut rester sur le dispositif lui-meme

## Verbatim ou mots emergents

- Mots choisis ou formules prononcees :
  - couche
  - voile
  - reserve
  - equilibre
  - contrepoids
- Citation courte eventuelle :
  - "Quand j'enleve cette couche, tout semble respirer autrement."

## Evaluation rapide

- Hesitation : faible
- Besoin de mediation orale : faible a moyen
- Retour a l'oeuvre : fort
- Potentiel du format : fort comme mediation low-tech

## Ce que nous avons appris

- apprentissage principal :
  - un dispositif tres simple peut rendre visibles les questions de couche, de voile et de reserve sans alourdir l'experience
- apprentissage secondaire :
  - le low-tech facilite souvent le retour a l'oeuvre parce que le support reste pauvre, lisible et peu intrusif
