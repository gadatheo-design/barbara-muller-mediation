# Fiche de session reelle 03 courte — Arduino Table de matieres

## Usage

Cette version courte est pensee pour une **prise de notes en 2 minutes** pendant ou juste apres une demonstration HKB du prototype `Table de matieres`. Elle permet de garder une trace minimale sur la lisibilite du geste tactile, le role de la lumiere et la capacite du dispositif a ouvrir un vocabulaire de regard sans ecran.
