# Fiche d'exemple de session de test 03 — Arduino Table de matieres

## Statut du document

Cette fiche est une **session d'exemple remplie** a partir de la documentation actuelle du prototype Arduino `Table de matieres`. Elle montre comment utiliser le gabarit de session pour une piste sans ecran, tactile et demonstrative. Elle ne doit pas etre lue comme une preuve terrain exhaustive.

## Identification

- Date : exemple de session a completer lors d'un vrai essai
- Lieu : demonstration HKB, table de mediation, ou test entre pairs
- Personne qui observe / medie : etudiant·e ou mediateur·trice du projet
- Format teste : prototype Arduino `Table de matieres`
- Version ou fichier concerne : `arduino-experiments/02_table_de_matieres/`
- Duree approximative : 3 a 5 minutes

## Question principale du test

- Que cherche-t-on a verifier dans cette session ?

Verifier si un dispositif tactile tres simple, sans ecran, aide a formuler des qualites picturales comme voile, trace, masse, reserve ou effacement, et s'il prepare utilement le regard vers une oeuvre ou une reproduction.

## Contexte

- Type de situation :
  - demonstration HKB
  - test entre pairs
- Nombre de personnes presentes :
  - 1 a 3
- Public concerne :
  - pair, enseignant, visiteur curieux, personne moins a l'aise avec les interfaces ecran
- Conditions particulieres :
  - table ou plaque en carton
  - mediation orale legere
  - reproduction ou image de reference placee a proximite

## Profil general du visiteur ou du groupe

- Age approximatif ou type de public :
  - adolescent ou adulte
- Familiarite avec art abstrait :
  - variable
- Familiarite avec smartphone / dispositif :
  - peu importante ici
- Commentaire utile :
  - le format rassure souvent les personnes qui hesitent face a une interface numerique plus chargee

## Deroulement

- Comment l'entree dans l'experience a ete introduite :
  - une phrase du type : "Touchez deux ou trois matieres et dites ce qu'elles vous font attendre du tableau."
- Le visiteur a-t-il compris rapidement ce qu'il fallait faire :
  - oui, l'action de toucher est immediate
- Le dispositif a-t-il demande une explication orale supplementaire :
  - oui, surtout pour relier les materiaux a des qualites picturales plutot qu'a des significations fixes
- A quel moment principal une hesitation est apparue :
  - au moment de comprendre que le prototype ne demande pas de "bonne reponse", mais une relation sensible entre matiere et regard
- A quel moment principal une comprehension ou un engagement est apparu :
  - lorsque deux materiaux contrastes, par exemple `papier calque` et `carton sombre`, sont compares a voix haute

## Observation qualitative

### Comprehension

- comprehension immediate / tardive / fragile :
  - immediate pour l'action, plus progressive pour l'intention de mediation
- ce qui a ete compris :
  - que chaque matiere ouvre une qualite de regard differente
- ce qui est reste flou :
  - le degre d'equivalence entre matiere et peinture ; il faut rappeler qu'il s'agit d'une analogie, pas d'une traduction exacte

### Rythme

- rythme trop rapide / juste / trop lent :
  - juste
- le visiteur a-t-il pris le temps :
  - oui, surtout si le dispositif est pose et non tenu a la main
- l'attention est-elle restee sur l'ecran, le materiau ou l'oeuvre :
  - sur le materiau d'abord, puis vers la reproduction ou l'oeuvre si celle-ci est explicitement reintroduite

### Geste ou action

- le geste ou l'action etait-il lisible :
  - tres lisible
- le changement produit etait-il perceptible :
  - oui, grace a la lumiere douce et au contraste entre les matieres
- le moment de passage a l'etape suivante etait-il clair :
  - oui, mais la suite depend de la mediation orale ou de la presence d'une reproduction

### Retour a l'oeuvre

- le retour a l'oeuvre a-t-il eu lieu :
  - oui, si une image ou une reproduction est placee a cote
- si oui, de quelle maniere :
  - le visiteur revient a l'oeuvre avec un petit vocabulaire deja active : voile, masse, reserve, douceur
- si non, pourquoi probablement :
  - si le prototype est montre seul, il peut rester au stade de la demonstration tactile

## Verbatim ou mots emergents

- Mots choisis ou formules prononcees :
  - voile
  - rugueux
  - masse
  - reserve
  - douceur
- Citation courte eventuelle :

## Ce que nous avons appris

- apprentissage principal :
  - retirer l'ecran deplace utilement la mediation vers le corps, la comparaison et la parole
- apprentissage secondaire :
  - le prototype gagne beaucoup si une reproduction ou une oeuvre est placee tout de suite a proximite
