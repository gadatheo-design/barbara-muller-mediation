# Introduction du laboratoire de médiation

## Statut

Document de cadrage pour la présentation finale HKB. Il formule le projet comme un laboratoire en cours plutôt que comme un dispositif définitivement résolu.

## Intention

Présenter le travail autour des œuvres de Barbara Müller comme une recherche de médiation. Le prototype mobile en fait partie, mais il ne résume pas à lui seul la démarche.

## Description

Le projet s'est développé comme une série d'expériences autour d'une même question : comment accompagner l'attention d'un visiteur face à des œuvres abstraites sans remplacer la rencontre avec l'œuvre par l'écran. Dans cette perspective, plusieurs formats ont été explorés : sketch Processing local, prototype web mobile en p5.js, parcours guidé, mode de test documenté, export JSON, scénarios Figma, pistes Arduino, projection, version tablette et alternatives low-tech.

L'ensemble peut être compris comme un laboratoire de médiation parce qu'il compare des formes d'adresse différentes : interaction individuelle, médiation accompagnée, documentation de test, installation collective et supports non numériques.

## Expérience visiteur

Selon le format proposé, le visiteur ne traverse pas la même expérience. Il peut manipuler un prototype sur téléphone, suivre un parcours guidé entre trois œuvres, tester une interaction plus technique, ou participer à une forme plus légère menée oralement. Le point commun reste le retour vers l'œuvre réelle : regarder plus lentement, repérer des couches, sentir des relations de couleur, puis vérifier ces impressions devant la peinture.

## Rôle du médiateur

Le médiateur ne sert pas uniquement à expliquer l'outil. Il observe, ajuste le rythme, choisit le bon format selon le contexte, et aide à replacer l'expérience numérique dans une rencontre avec l'œuvre. Dans ce laboratoire, la médiation humaine reste centrale.

## Ce que nous avons appris

Le travail montre qu'un même projet peut produire plusieurs formes utiles sans devoir être un produit complet. Il a aussi confirmé qu'en médiation, les prototypes les plus intéressants ne sont pas forcément les plus complexes techniquement : la clarté du geste, le rythme de l'attention et la qualité du retour vers l'œuvre comptent davantage.

## Limites actuelles

Le laboratoire n'aboutit pas encore à un format unique validé pour une mise en exposition durable. Certaines pistes restent conceptuelles, d'autres sont partiellement implémentées, et les tests publics restent limités en nombre.

## Développements possibles

La suite consiste moins à ajouter des fonctions qu'à mieux articuler les formats existants : version visiteur très simple, version médiateur plus contrôlée, documentation de test plus solide, et éventuelle traduction en installation ou en format pédagogique non numérique.
