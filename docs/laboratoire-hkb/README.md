# Documentation laboratoire HKB

Ce dossier rassemble la documentation de présentation du projet comme **laboratoire de médiation** autour des œuvres de Barbara Müller. L'enjeu n'est pas de présenter une application unique et stabilisée, mais un ensemble de prototypes, de tests et de pistes qui explorent différentes façons d'accompagner le regard.

## Comment utiliser ce dossier pendant la présentation HKB

Commencer par [00_INTRODUCTION_LABORATOIRE.md](./00_INTRODUCTION_LABORATOIRE.md), puis poursuivre avec [01_CARTE_DES_PROTOTYPES.md](./01_CARTE_DES_PROTOTYPES.md) pour situer l'ensemble. Ensuite, choisir les fichiers en fonction du format de soutenance :

- pour montrer le prototype accessible au public : [03_P5JS_WEB.md](./03_P5JS_WEB.md), [04_PARCOURS_GUIDE.md](./04_PARCOURS_GUIDE.md), [10_QR_CODE_MEDIATION.md](./10_QR_CODE_MEDIATION.md)
- pour montrer la dimension expérimentale et technique : [02_PROCESSING_LOCAL.md](./02_PROCESSING_LOCAL.md), [05_MODE_TEST_EXPORT_JSON.md](./05_MODE_TEST_EXPORT_JSON.md), [06_PISTES_FIGMA.md](./06_PISTES_FIGMA.md), [07_PISTES_ARDUINO.md](./07_PISTES_ARDUINO.md)
- pour ouvrir vers l'installation et la médiation élargie : [08_PROJECTION_INSTALLATION.md](./08_PROJECTION_INSTALLATION.md), [09_VERSION_TABLETTE.md](./09_VERSION_TABLETTE.md), [11_LOW_TECH.md](./11_LOW_TECH.md)
- pour conclure de manière réflexive : [12_PISTES_ABANDONNEES_ET_LIMITES.md](./12_PISTES_ABANDONNEES_ET_LIMITES.md), [13_ROADMAP_FUTURE.md](./13_ROADMAP_FUTURE.md)

## Lecture rapide suggérée

1. Introduction du laboratoire
2. Carte des prototypes
3. Prototype p5.js web
4. Parcours guidé
5. Mode test et export JSON
6. Une ou deux pistes d'extension selon le temps disponible
7. Limites et roadmap

Pour relier cette lecture aux decisions du projet :

- utiliser [COMPARATIVE_MATRIX.md](../COMPARATIVE_MATRIX.md) pour comparer les formats
- utiliser [TEST_SYNTHESIS_TEMPLATE.md](../TEST_SYNTHESIS_TEMPLATE.md) pour relire les essais
