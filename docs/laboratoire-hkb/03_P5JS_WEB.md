# p5.js web

## Statut

Prototype public principal à ce stade. Il s'agit de la version la plus accessible pour des tests sur smartphone et pour une diffusion légère.

## Intention

Proposer une expérience mobile web facilement partageable afin de tester la médiation dans des conditions proches d'une visite réelle, sans installation lourde.

## Description

La version p5.js web transpose le projet dans un format mobile-first. Elle peut être publiée via GitHub Pages et ouverte directement dans le navigateur d'un téléphone. Ce choix technique permet de diffuser un prototype sans procédure d'installation pour le public.

Le web n'est pas présenté ici comme un aboutissement, mais comme une surface de test particulièrement utile : elle rend visibles les choix de rythme, de navigation, de langage et de lisibilité sur petit écran.

## Expérience visiteur

Le visiteur peut accéder au prototype sur son propre smartphone, suivre une interaction courte, choisir des mots, revenir vers l'œuvre, puis éventuellement consulter un résumé de test. L'expérience reste légère, immédiate et compatible avec un accès par QR code.

## Rôle du médiateur

Le médiateur peut introduire le prototype, aider à formuler l'entrée dans l'expérience et observer les moments d'hésitation. Il peut aussi utiliser cette version comme outil de test comparatif, en regardant ce qui fonctionne ou non sur téléphone.

## Ce que nous avons appris

Le passage au web a rendu le projet beaucoup plus testable dans des conditions réalistes. Il a aussi obligé à simplifier certaines interactions, à clarifier les consignes et à travailler la lisibilité sur smartphone.

## Limites actuelles

Le téléphone reste un support contraint : écran réduit, attention fragmentée, dépendance à la batterie, au réseau et à la qualité de l'appareil. Certaines nuances visuelles ou certains gestes se lisent moins bien que dans un format local plus contrôlé.

## Développements possibles

Cette version peut continuer à servir de base pour des tests publics, des itérations rapides et une diffusion simple. Elle peut également nourrir une future version tablette, un usage en kiosque ou une déclinaison encore plus minimale pour des contextes d'exposition.
