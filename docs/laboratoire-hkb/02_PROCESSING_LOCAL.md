# Processing local

## Statut

Prototype expérimental local, utile pour la recherche et le réglage des interactions. Ce format n'est pas pensé en priorité pour un accès public autonome.

## Intention

Utiliser un sketch local comme terrain d'essai rapide pour les comportements visuels, les rythmes d'apparition, les états de regard et les réponses sensibles à l'interaction.

## Description

La version Processing constitue un sketch expérimental local. Elle permet de tester des interactions visuelles de manière directe, sans dépendre d'un navigateur mobile ni d'une diffusion en ligne. Ce cadre est particulièrement utile pour explorer la relation entre geste, temporalité et transformation de l'image.

Cette piste est moins orientée vers le public final que vers l'atelier de prototypage. Elle peut aussi servir de base pour des formats plus installatifs, par exemple une projection ou une station de médiation située.

## Expérience visiteur

Dans sa forme actuelle, l'expérience visiteur y est plus proche d'un prototype de laboratoire que d'un outil autonome. Elle peut fonctionner dans un contexte accompagné, où le visiteur découvre une interaction préparée par le médiateur, mais elle demande souvent une mise en place plus précise.

## Rôle du médiateur

Le médiateur ou la personne qui anime le test joue ici un rôle important. Il lance le sketch, choisit les paramètres pertinents et guide l'attention du visiteur. Dans ce format, la médiation ne disparaît pas derrière l'interface ; elle cadre explicitement l'expérience.

## Ce que nous avons appris

Processing a été utile pour tester rapidement des hypothèses visuelles : opacités, traces, rythmes, transitions et modes de retour vers l'œuvre. Ce cadre a facilité un prototypage sensible avant le passage vers une version web plus accessible.

## Limites actuelles

Le sketch local reste moins public-facing qu'une version web. Son accès dépend d'une machine dédiée et d'une installation technique préalable. Il est donc moins adapté à une circulation légère dans un contexte d'exposition ouverte.

## Développements possibles

Cette base pourrait être prolongée vers une version de projection ou vers une installation plus située dans l'espace. Elle reste pertinente comme environnement d'essai pour préparer de futures formes d'interaction plus stables.
