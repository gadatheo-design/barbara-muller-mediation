# Fiche low-tech — Mots sensibles

## Correspondance numérique
Choix de mot + phrase finale.

## Matériel
- Cartes-mots.
- Petite table ou mur magnétique.
- Reproduction de l’œuvre.

## Expérience
Après observation, le public choisit un mot et le place près de l’œuvre.

## Exemples
Œuvre 1 : voile, poids, appui, tension, seuil, suspension.  
Œuvre 2 : masse, chaleur, contraste, bloc, voisinage, domination.  
Œuvre 3 : trace, souffle, silence, effacement, fragilité, apparition.

## Objectif de médiation
Faire passer le regard vers le langage sans imposer une interprétation.
