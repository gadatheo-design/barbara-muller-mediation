# Barbara Müller — Prototype Processing V6 FINAL

Cette version compile la V5 et les modules avancés proposés pour aller plus loin.

## Nouveautés V6

### 1. Mode sans texte
- Touche `I`
- Interface très minimale : presque plus de consignes, seulement des repères.

### 2. Timer de regard lent
- Intégré au mode “silence” de l’œuvre 3.
- L’image se révèle mieux si la personne reste vraiment un moment.

### 3. Micro-questions contextuelles
- Questions courtes qui changent toutes les 7 secondes.
- FR / DE / EN selon la langue active.

### 4. Trace fantôme du mouvement
- Touche `Q`
- Le mouvement de la souris / du doigt laisse une trace fine.

### 5. Respiration de l’interface
- L’opacité de l’interface varie subtilement.

### 6. Feedback visuel du mot choisi
- Quand un mot sensible est choisi, un cercle se propage depuis le centre.

### 7. Déplacement de l’attention
- `N` = point avant
- `F` = point après
- `V` = afficher avant/après
- Le prototype formule si le regard s’est peu, moyennement ou fortement déplacé.

### 8. Cartel poétique
- Touche `Y`
- Court cartel sensible adapté à chaque œuvre.

### 9. Mode score / partition de regard
- Touche `U`
- Une consigne de regard change toutes les 8 secondes.

### 10. Reset doux
- `R` lance un fondu blanc avant de réinitialiser.

## Rappels V5 conservés

- `K` : mode kiosk / exposition
- `G` : archive visuelle finale PNG
- `B` : langue FR / DE / EN
- `D` : dashboard médiateur
- `X` : export CSV des points de regard
- `T` : retour silencieux à l’œuvre
- `W` : mots ON/OFF
- `M` : carte du regard ON/OFF
- `L` : résumé session TXT
- `S` : capture

## Test rapide recommandé

1. Lancer le sketch.
2. Appuyer sur `B` pour choisir la langue.
3. Appuyer sur `I` pour tester le mode minimal.
4. Appuyer sur `Y` pour afficher le cartel poétique.
5. Marquer un point initial avec `N`.
6. Tester une interaction.
7. Marquer un point final avec `F`.
8. Afficher le déplacement avec `V`.
9. Choisir un mot.
10. Générer l’archive avec `G`.

## Direction conceptuelle

Cette V6 transforme le prototype en partition d’attention :
observer, attendre, toucher, nommer, comparer, revenir à l’œuvre.
