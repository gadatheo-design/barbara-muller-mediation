# Fiche d'exemple de session de test 02 — p5 Carre de couleur fixe

## Statut du document

Cette fiche est une **session d'exemple remplie** pour une experience p5.js du laboratoire public. Elle s'appuie sur la documentation existante du sketch `fixed_colour_square` et montre comment documenter un essai bref d'une piste experimentale plus focalisee que le prototype principal.

## Identification

- Date : exemple de session a completer lors d'un vrai essai
- Lieu : demonstration HKB, test entre pairs, ou consultation du laboratoire public web
- Personne qui observe / medie : etudiant·e ou mediateur·trice du projet
- Format teste : experience web `Carre de couleur fixe`
- Version ou fichier concerne : `web/laboratoire/experiments/01_fixed_colour_square/`
- Duree approximative : 2 a 4 minutes

## Question principale du test

- Que cherche-t-on a verifier dans cette session ?

Verifier si une experience tres courte, sans image d'oeuvre explicite, peut faire sentir la perception contextuelle de la couleur et preparer un vocabulaire de mediation autour du voisinage chromatique.

## Contexte

- Type de situation :
  - demonstration HKB
  - test entre pairs
- Nombre de personnes presentes :
  - 1 a 3
- Public concerne :
  - jury, pair, visiteur curieux, personne voyant le laboratoire public
- Conditions particulieres :
  - temps court
  - ecran d'ordinateur ou telephone
  - mediation orale tres legere

## Profil general du visiteur ou du groupe

- Age approximatif ou type de public :
  - adulte
- Familiarite avec art abstrait :
  - variable
- Familiarite avec smartphone / dispositif :
  - generalement bonne
- Commentaire utile :
  - ce format peut etre compris meme par une personne qui ne connait pas encore le prototype principal

## Deroulement

- Comment l'entree dans l'experience a ete introduite :
  - une phrase du type : "Le carre ne change jamais. Regardez comment c'est le fond qui modifie votre perception."
- Le visiteur a-t-il compris rapidement ce qu'il fallait faire :
  - oui, le geste de glisser le carre est immediat
- Le dispositif a-t-il demande une explication orale supplementaire :
  - oui, mais tres courte, pour formuler l'enjeu perceptif
- A quel moment principal une hesitation est apparue :
  - au niveau de l'interpretation : comprendre que la demonstration ne parle pas du carre lui-meme, mais de la relation entre couleur fixe et contexte
- A quel moment principal une comprehension ou un engagement est apparu :
  - lorsqu'une meme couleur semble visiblement differente sur deux champs de fond

## Observation qualitative

### Comprehension

- comprehension immediate / tardive / fragile :
  - immediate pour l'action, plus lente pour l'idee de mediation
- ce qui a ete compris :
  - que le carre garde la meme couleur et que le voisinage agit
- ce qui est reste flou :
  - le lien exact avec les oeuvres si l'experience est montree seule, sans commentaire

### Rythme

- rythme trop rapide / juste / trop lent :
  - juste
- le visiteur a-t-il pris le temps :
  - oui, parce que le geste est minimal et peut etre repete
- l'attention est-elle restee sur l'ecran, le materiau ou l'oeuvre :
  - surtout sur l'ecran, mais au service d'une idee perceptive simple qui peut ensuite etre reconnectee a l'oeuvre

### Geste ou action

- le geste ou l'action etait-il lisible :
  - tres lisible
- le changement produit etait-il perceptible :
  - oui, surtout grace au halo et au passage d'une zone a l'autre
- le moment de passage a l'etape suivante etait-il clair :
  - comme il s'agit d'un sketch autonome, la sortie doit etre apportee par la mediation orale

### Retour a l'oeuvre

- le retour a l'oeuvre a-t-il eu lieu :
  - pas automatiquement
- si oui, de quelle maniere :
  - seulement si le mediateur relie explicitement l'experience a une oeuvre ou a la question du voisinage chromatique
- si non, pourquoi probablement :
  - ce sketch est une experience focale de laboratoire, pas un parcours complet

## Verbatim ou mots emergents

## Evaluation rapide

- Hesitation : faible
- Besoin de mediation orale : faible a moyen
- Retour a l'oeuvre : faible en autonomie, moyen si accompagne
- Potentiel du format : fort comme sketch de laboratoire

## Ce que nous avons appris

- apprentissage principal :
  - une experience tres simple peut rendre visible une hypothese de mediation plus nettement qu'un prototype plus complexe
- apprentissage secondaire :
  - sans mise en relation orale avec les peintures, le sketch reste plus demonstratif que pleinement mediateur
